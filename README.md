# JS Git Demo

A quick in-browser demo of cloning repos.

It can be a little CPU-hungry, as a heads up. Eventually
it'll use web workers.

## Usage

```bash
$ git clone https://github.com/Dignifiedquire/js-git-demo.git
$ cd js-git-demo
$ npm install
$ npm start
$ open http://localhost:9999
```
